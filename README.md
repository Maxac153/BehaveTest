# PythonBehave

## Описание

Цель проекта: получение навыков в написание автоматизированных тестов (BDD) на Python<br>
на примере веб-сайта (<a href="http://users.bugred.ru/">users</a>).

## Запуск тестов

Для запуска тестов нужно ввести следующую команду</br>
behave

## Технологии

- Python 3.11
- Behave
